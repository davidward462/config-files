# config-files
Useful linux configuration files

This repository is for backup and reference purposes. There is no installer script.
You will need to manually copy files over to their required locations if you clone this repo.

## nvim/init.lua
Copied and slightly altered from https://github.com/radleylewis/nvim-lite/blob/master/init.lua
Generally needs to be stored in ~/.config/nvim

## vimrc
This would be named .vimrc (hidden file) in a system. It contains some simple and useful settings for vim (or neovim). 
Usually would be stored in the home directory.
